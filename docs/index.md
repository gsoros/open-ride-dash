# OpenRideDash Documentation Hub

Welcome to the official documentation for **OpenRideDash** (ORD) - an open, modular firmware and hardware platform for ESP32-based e-bike displays communicating over CAN, with optional BLE connectivity and mobile app integration.

## Quick Start

If you're new to OpenRideDash, start here:

1. **[Overview](overview.md)** - Understand what OpenRideDash is and what it can do
2. **[System Architecture](architecture/system-architecture.md)** - Learn about the overall system design
3. **[Getting Started](getting-started.md)** - Set up your development environment
4. **[Phase 1: Core System](phases/phase1-core.md)** - Build the minimum viable product
