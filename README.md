## 🚀 What is OpenRideDash?

OpenRideDash (ORD) is a comprehensive open-source platform for creating customizable e-bike displays and controllers. Built around ESP32 microcontrollers with CAN bus communication, it provides:

- **CAN Bus Integration** - Communicate with e-bike motor controllers (Bafang, etc.)
- **BLE Connectivity** - Connect to mobile apps and external sensors
- **Modular Firmware** - Hardware-abstracted, easy to extend
- **Mobile App** - Configuration, telemetry, and data logging
- **OTA Updates** - Wireless firmware updates
